📘 Day 07: Modules and Packages in Python

Modules and packages help organize Python programs into reusable components. They make code cleaner, modular, and easier to maintain.

📖 Introduction to Modules and Packages

As programs become larger, writing everything in a single file becomes difficult to manage.

Python provides Modules and Packages to organize code into smaller, reusable files.

Benefits

  • Reusable code
  • Better organization
  • Easier maintenance
  • Improves readability
  • Encourages modular programming

📦 What is a Module?

A module is simply a Python file (.py) containing variables, functions, and classes.

Python provides many built-in modules such as:

  • math
  • random
  • datetime
  • os
  • sys

Example

import math

Now we can access all functions inside the math module.

print(math.sqrt(16))

Output

4.0

📥 Importing Modules

Import an entire module.

import math

print(math.sqrt(16))

Output

4.0

Another example

import numpy

Create a NumPy array.

arr = numpy.array([1,2,3,4,5])

print(arr)

🎯 Importing Specific Functions

Instead of importing the whole module, import only the required functions.

from math import sqrt, pi

Now they can be used directly.

print(sqrt(16))

print(pi)

Output

4.0

3.141592653589793

This makes the code shorter.

🏷️ Using Aliases

Aliases provide shorter names for modules.

import numpy as np

Example

numbers = np.array([1,2,3,4,5])

print(numbers)

Output

[1 2 3 4 5]

Another common alias

import pandas as pd

⭐ Importing All Functions

You can import every function from a module.

from math import *

Example

print(sqrt(16))

print(pi)

Output

4.0

3.141592653589793

Note: Although possible, importing everything using * is not recommended because it can lead to namespace conflicts.

🛠️ Creating Custom Packages

A package is a collection of related modules organized inside a directory.

Example folder structure

package/
│
├── __init__.py
├── maths.py
│
└── subpackages/
    ├── __init__.py
    └── mult.py

maths.py

def addition(a, b):
    return a + b

mult.py

def multiply(a, b):
    return a * b

📂 Importing Custom Modules

Import a Specific Function

from package.maths import addition

print(addition(2,3))

Output

5

Import the Entire Module

from package import maths

print(maths.addition(2,3))

Output

5

Import Everything

from package.maths import *

print(addition(2,3))

Output

5

Import from a Subpackage

from package.subpackages.mult import multiply

print(multiply(4,5))

Output

20

📁 Package Structure

Example

project/

│── main.py

│
└── package/

    ├── __init__.py

    ├── maths.py

    │
    └── subpackages/

        ├── __init__.py

        └── mult.py

Here,

  • main.py imports functions from the package.
  • __init__.py tells Python that the directory should be treated as a package.

❌ Common Errors

Module Not Found

import maths

Raises

ModuleNotFoundError

when the module does not exist.

Misspelled Module Name

Incorrect

import mat

Correct

import math

Using Functions Without Importing

Incorrect

print(sqrt(16))

Raises

NameError

Correct

from math import sqrt

print(sqrt(16))

Using import *

Although valid,

from math import *

is discouraged because multiple modules may contain functions with the same name.

✅ Best Practices

  • Import only what you need.
  • Use aliases for long module names.
  • Avoid from module import *.
  • Organize large projects into packages.
  • Give modules meaningful names.
  • Keep related functions together in the same module.

📖 Introduction to the Standard Library

Python's Standard Library is a vast collection of modules and packages that come pre-installed with Python.

These modules help developers perform tasks such as:

  • Mathematical calculations
  • Working with files
  • Handling JSON and CSV data
  • Managing directories
  • Working with dates and time
  • Generating random values
  • Pattern matching using Regular Expressions

Note: Since these modules are built into Python, they do not require installation using pip.

📦 The array Module

The array module provides a space-efficient way to store arrays of similar data types.

import array

arr = array.array('i', [1, 2, 3, 4])

print(arr)

Output

array('i', [1, 2, 3, 4])

Here,

  • 'i' represents an integer array.

➗ The math Module

The math module provides mathematical functions.

import math

print(math.sqrt(16))

Output

4.0

Other useful functions include:

print(math.pi)

print(math.factorial(5))

print(math.ceil(3.2))

print(math.floor(3.8))

🎲 The random Module

The random module generates random values.

import random

print(random.randint(1, 10))

Output

7

Note: randint() returns a random integer between the given range (inclusive).

Choose a random item.

print(random.choice(["apple", "banana", "cherry"]))

Example Output

banana

📂 The os Module

The os module provides functions to interact with the operating system.

Current working directory

import os

print(os.getcwd())

Create a new directory

os.mkdir("test_dir")

Other useful functions

os.listdir()

os.remove("sample.txt")

os.rmdir("test_dir")

📁 The shutil Module

The shutil module performs high-level file operations.

Copy a file

import shutil

shutil.copyfile("source.txt", "destination.txt")

Other useful functions

shutil.move()

shutil.copytree()

shutil.rmtree()

🌐 The json Module

JSON is widely used for storing and exchanging data.

Convert Dictionary to JSON

import json

data = {
    "name": "Prav",
    "age": 25
}

json_str = json.dumps(data)

print(json_str)

print(type(json_str))

Output

{"name": "Prav", "age": 25}

<class 'str'>

Convert JSON to Dictionary

parsed_data = json.loads(json_str)

print(parsed_data)

print(type(parsed_data))

Output

{'name': 'Prav', 'age': 25}

<class 'dict'>

📄 The csv Module

The csv module helps read and write CSV files.

Writing to a CSV File

import csv

with open("example.csv", mode="w", newline="") as file:

    writer = csv.writer(file)

    writer.writerow(["Name", "Age"])

    writer.writerow(["Prav", 32])

Reading a CSV File

with open("example.csv", mode="r") as file:

    reader = csv.reader(file)

    for row in reader:
        print(row)

Output

['Name', 'Age']

['Prav', '32']

📅 The datetime Module

The datetime module works with dates and time.

from datetime import datetime, timedelta

current_date = datetime.now()

print(current_date)

Yesterday's date

yesterday = current_date - timedelta(days=1)

print(yesterday)

Tomorrow's date

tomorrow = current_date + timedelta(days=1)

print(tomorrow)

⏱️ The time Module

The time module provides time-related functions.

import time

print(time.time())

Pause execution

time.sleep(2)

print("Program Resumed")

sleep(2) pauses the program for 2 seconds.

🔍 The re Module (Regular Expressions)

The re module is used for searching and matching patterns in text.

import re

pattern = r"\d+"

text = "There are 123 apples and 456 oranges."

match = re.search(pattern, text)

print(match.group())

Output

123

Find all numbers

numbers = re.findall(pattern, text)

print(numbers)

Output

['123', '456']

❌ Common Errors

Confusing json.dump() with json.dumps()

json.dumps() converts an object to a JSON string, while json.dump() writes JSON to a file object and requires a file as its second argument. Calling json.dump(data) alone raises a TypeError.

Incorrect (missing the file argument)

json.dump(data)

Correct (get a JSON string)

json_string = json.dumps(data)

Correct (write directly to a file)

with open("data.json", "w") as f:
    json.dump(data, f)

Forgetting to Import a Module

print(math.sqrt(16))

Raises

NameError

Correct

import math

Incorrect CSV Writer Variable

Incorrect

write.writerow(...)

Correct

writer.writerow(...)

Incorrect Variable Name

Incorrect

yesterday = now - timedelta(days=1)

Correct

yesterday = current_date - timedelta(days=1)

✅ Best Practices

  • Import only the modules you need.
  • Use aliases when appropriate (e.g., import numpy as np).
  • Use with open() for file handling.
  • Use json.dumps() and json.loads() correctly.
  • Close files automatically using context managers.
  • Use descriptive variable names.
