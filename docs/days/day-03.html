📖 Introduction to Operators

Operators are symbols that tell Python to perform a specific operation on one or more operands.

Python provides several types of operators:

  • Arithmetic Operators
  • Comparison Operators
  • Logical Operators

These operators are widely used in calculations, conditions, and decision-making.

➕ Arithmetic Operators

Arithmetic operators perform mathematical calculations.

Operator Name Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
// Floor Division a // b
% Modulus a % b
** Exponentiation a ** b

Example

a = 15
b = 20

add_result = a + b
sub_result = b - a
multi_result = a * b
div_result = a / b
floor_div_result = a // b
modulus_result = a % b
exponent_result = a ** b

print("Addition:", add_result)
print("Subtraction:", sub_result)
print("Multiplication:", multi_result)
print("Division:", div_result)
print("Floor Division:", floor_div_result)
print("Modulus:", modulus_result)
print("Exponentiation:", exponent_result)

Output

Addition: 35
Subtraction: 5
Multiplication: 300
Division: 0.75
Floor Division: 0
Modulus: 15
Exponentiation: 332525673007965087890625

Note: Floor division (//) returns the integer quotient by discarding the decimal part.

Addition (+)

Adds two operands.

a = 10
b = 5

print(a + b)

Output

15

Subtraction (-)

Subtracts one operand from another.

print(20 - 8)

Output

12

Multiplication (*)

Multiplies two numbers.

print(6 * 7)

Output

42

Division (/)

Returns the quotient as a floating-point number.

print(20 / 5)

Output

4.0

Floor Division (//)

Returns only the integer quotient.

print(20 // 3)

Output

6

Modulus (%)

Returns the remainder after division.

print(20 % 3)

Output

2

Exponentiation (**)

Raises the first operand to the power of the second.

print(2 ** 5)

Output

32

⚖️ Comparison Operators

Comparison operators compare two values and always return True or False.

Operator Description
== Equal to
!= Not equal to
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to

Example

a = 10
b = 20

print(a == b)
print(a != b)
print(a > b)
print(a < b)
print(a >= b)
print(a <= b)

Output

False
True
False
True
False
True

Comparing Strings

str1 = "prav"
str2 = "kosanam"

print(str1 == str2)
print(str1 != str2)

Output

False
True

🔀 Logical Operators

Logical operators combine multiple conditions.

Operator Description
and Returns True if both conditions are True
or Returns True if at least one condition is True
not Reverses the Boolean value

Example

X = True
Y = False

result = X and Y
print(result)

result2 = X or Y
print(result2)

result3 = not X
print(result3)

Output

False
True
False

Truth Table

True True True True False
True False False True False
False True False True True
False False False False True

📌 Operator Precedence

Python follows operator precedence while evaluating expressions.

Highest precedence:

  1. () Parentheses
  2. ** Exponentiation
  3. *, /, //, %
  4. +, -
  5. Comparison Operators
  6. not
  7. and
  8. or

Example

result = 10 + 2 * 5

print(result)

Output

20

Using parentheses:

result = (10 + 2) * 5

print(result)

Output

60

❌ Common Errors

1. Division by Zero

10 / 0

Raises:

ZeroDivisionError

2. Comparing Different Data Types

10 > "20"

Raises:

TypeError

3. Incorrect Logical Expression

True and

Raises a syntax error.

4. Using Assignment Instead of Comparison

Incorrect

if a = 10:

Correct

if a == 10:

✅ Best Practices

  • Use meaningful variable names.
  • Use parentheses to improve readability.
  • Avoid unnecessary complex expressions.
  • Always check for division by zero.
  • Use comparison operators carefully inside conditions.

Example

if age >= 18 and is_verified:
    print("Access Granted")
