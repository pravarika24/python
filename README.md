# 🐍 Learn Python in 10 Days

A structured, beginner-friendly **10-day journey** through the Python programming language — from basic syntax to Object-Oriented Programming. Each day contains **hands-on Jupyter notebooks** (`.ipynb`) paired with **detailed notes** (`.md`) that explain the concepts, show examples, and include practice exercises.

---

## 📚 Curriculum

| Day | Topic | Folder | What you'll learn |
|:---:|-------|--------|-------------------|
| 01 | **Basics** | [Day_01_basics](Day_01_basics) | Syntax & semantics, comments, indentation, case sensitivity, line continuation, type inference |
| 02 | **Variables & Data Types** | [Day_02_Variables and Datatypes](Day_02_Variables%20and%20Datatypes) | Declaring variables, naming conventions, type checking & conversion, dynamic typing |
| 03 | **Operators** | [Day_03_operators](Day_03_operators) | Arithmetic, comparison, logical operators and operator precedence |
| 04 | **Control Flow** | [Day_04_pythoncontrolflow](Day_04_pythoncontrolflow) | Conditional statements (`if`/`elif`/`else`), `for` & `while` loops, `break`/`continue`/`pass` |
| 05 | **Data Structures** | [Day_05_DataStructures](Day_05_DataStructures) | Lists, tuples, dictionaries, slicing, and real-world examples |
| 06 | **Functions** | [Day_06_Functions](Day_06_Functions) | Defining functions, arguments, return values, `lambda`, `map()`, `filter()` |
| 07 | **Modules & Packages** | [Day_07_importing_creating_modules_and_packages](Day_07_importing_creating_modules_and_packages) | Importing, creating custom modules/packages, and the standard library |
| 08 | **File Handling** | [Day_08_Filehandling](Day_08_Filehandling) | Reading/writing text & binary files, file modes, working with paths |
| 09 | **Exception Handling** | [Day_09_exception_handling](Day_09_exception_handling) | `try`/`except`/`else`/`finally`, handling multiple exceptions, raising errors |
| 10 | **Object-Oriented Programming** | [Day_10_oops](Day_10_oops) | Classes & objects, inheritance, polymorphism, encapsulation & abstraction, magic methods, operator overloading |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/pravarika24/python.git
cd python
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv myenv
source myenv/bin/activate      # On Windows: myenv\Scripts\activate
```

### 3. Install the dependencies

```bash
pip install -r requirements.txt
pip install notebook            # to run the Jupyter notebooks
```

### 4. Launch Jupyter and start learning

```bash
jupyter notebook
```

Open any day's `.ipynb` file and run the cells, or read the accompanying `.md` notes.
